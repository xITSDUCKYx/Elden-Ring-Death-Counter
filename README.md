# Elden Ring Death Counter

## Overview

This is site made to log number of deaths per boss in elden ring.  
You can purchase the game here on [steam](https://store.steampowered.com/app/1245620/ELDEN_RING/)!


## Features 

* Increment deaths
* Logs deaths (per Boss)
* No dependencies
